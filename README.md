# 🚀 SEO Analyzer

A full-stack SEO analysis tool that checks website SEO health.

## Features

- SEO title check
- Meta description check
- Heading analysis
- Image alt check
- Technical SEO analysis
- Lighthouse performance report
- SEO score calculation
- Report history

## Tech Stack

Frontend:
- React.js
- Vite
- CSS

Backend:
- Node.js
- Express.js
- MongoDB
- Puppeteer
- Lighthouse
